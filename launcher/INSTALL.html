This guide takes you from a fresh Mac to a working Bidnamic OS chat prompt. Work through the sections in order — each one builds on the last. Sections 2–5 prepare your Mac; sections 6–8 get you connected.

1.Before you start

A few things to have ready. Nothing here is hard — just easier with everything to hand.

You will need

  • Your Mac admin password (the one you use to log in / install software)
  • Your work email address
  • A stable internet connection
  • Your Mac plugged into power

You'll spend a lot of this guide in the Terminal app. To open it at any point: press Command + Space, type Terminal, and press Enter. (It also lives in Applications → Utilities in the Finder.) When a step gives you a command, you can hover the grey box and click copy, then paste into Terminal with Command + V and hit Enter.

2.Upgrade to macOS Tahoe Mac setup

Bidnamic OS expects an up-to-date Mac. Start here even if you think you're current.

  1. Click the Apple logo in the top-left corner of your screen.
  2. Click System Settings.
  3. Go to General.
  4. Click Software Update and wait for your Mac to check for updates.
  5. If macOS Tahoe is shown, click Upgrade Now (or Update Now) and follow the on-screen instructions.
Keep it plugged in. Don't close the lid while the update runs. Your Mac may restart several times — that's normal. If macOS Tahoe doesn't appear at all, your Mac may already be up to date, or it may not be compatible — ask for help if you're unsure.

3.Install Xcode Command Line Tools Mac setup

These are the developer tools Homebrew needs. Open Terminal (Command + Space, type Terminal, Enter), then run:

xcode-select --install

A pop-up should appear — click Install, follow the prompts, and wait for it to finish. If Terminal says the tools are already installed, that's fine; carry on.

Check it worked

In the same Terminal window, run:

clang --version

If the tools installed correctly you'll see a version message — something like:

Apple clang version ...
Target: ...
Thread model: ...

If instead you see command not found, run xcode-select --install again, let it finish, then re-run the clang --version check.

4.Install Homebrew Mac setup

Homebrew is the installer we use to deliver Bidnamic OS.

  1. Open your web browser and go to this link to download the installer: 
    https://github.com/Homebrew/brew/releases/download/5.1.14/Homebrew.pkg
  2. Open the downloaded Homebrew.pkg file and follow the installer prompts.
  3. When asked, enter your Mac admin password, then wait for it to finish.
  4. Quit and reopen Terminal once the installer is done (close it if it's open, then open it again).

Check it installed correctly:

brew --version

A Homebrew version number means you're good. If Terminal says brew: command not found, restart your Mac and try the command again.

5.Install Tailscale Mac setup

Tailscale is the secure network that lets Bidnamic OS reach our systems. You'll need it running and signed in.

  1. Open your web browser and go to: 
    https://tailscale.com/download/mac
  2. Download the macOS app, open the downloaded file, and follow the installation prompts.
  3. Open Tailscale from your Applications folder.
  4. Click Log in (or Sign in) and sign in using your work email address. Follow any browser prompts to complete sign-in.
  5. If macOS asks for permission to add VPN configurations, click Allow.
Account approval. Your Tailscale account may need approving before it connects. If you sign in and it doesn't connect, ask for help and we'll approve you.
Mac setup complete. macOS Tahoe, Xcode Command Line Tools, Homebrew, and Tailscale are now all in place. The rest of this guide gets you connected to Bidnamic OS.

6.Set up your accounts Connect

Bidnamic OS signs you in through Claude.ai. Sort these out before you try to connect — they each arrive as an email invite.

  1. Accept your AWS Identity Center invitation email and set up 2FA (two-factor authentication) when prompted.
  2. Accept your invite to claude.ai. Once accepted, claude.ai will prompt you to download the Claude app — you can ignore this and close the tab.
  3. Log in to claude.ai and check everything works — you should land in a working chat.
  4. Double-check Tailscale is running and connected (look for it in the menu bar at the top of your screen). If your account still needs approving, ask for help.

7.Install Bidnamic OS Connect

Open the Terminal app — it's in Applications → Utilities in the Finder, or press Command + Space and type Terminal. Run these four commands in order, pressing Enter after each and letting it finish before the next.

1 — Add the AWS tap

brew tap aws/aws

2 — Add the Bidnamic tap

brew tap bidnamic/homebrew-tap

3 — Install Bidnamic OS

brew install bidnamic-os

4 — Run the post-install step

bidnamic-os post-install

That last command finishes wiring everything up. Once it completes, you're ready to connect.

8.Connect for the first time Connect

Now start Bidnamic OS:

bidnamic-os

Once it connects, pick your preferred terminal style by typing a number from 1 to 4 and pressing Enter.

AWS sign-in. The first time you run it, you'll be asked to authenticate with the credentials you set up on AWS Identity Center. It'll show you a code in your Terminal and open a browser — confirm that the code in the browser matches the one in your Terminal, then approve.

After that, the system walks you through a one-time Claude.ai login. Here's what to expect:

  1. When prompted, select option 1 to log in through Claude.ai.
  2. Copy the link it shows you into your web browser.
  3. In the browser, click Authorise.
  4. Copy the auth code the browser gives you, paste it back into Terminal, and hit Enter.
  5. Finish setting your preferences when asked.
You're in. You should land at the Bidnamic OS chat prompt. Type hello and hit Enter — a reply within a second or two means everything's working. From now on, you just run bidnamic-os to start a session.
